As configurações de e-mails são usadas para todas as atividades de envio de e-mail do Mango.

SMTP host é o nome do servidor onde o MTA (Mail Transfer Agent ou servidor de email) é executado. Se esse servidor requer authenticação, selecione a caixa de seleção Use authorization, e preencha os campos Username e Password.

O campo 'From' address representa o endereço que será usado como rementente em cada email. O campo 'From' name é um campo opcional para especificar um alias — ou um nome próprio — que será exibido como o remetente do email.

É possível testar as configurações clicando no ícone , que tentará enviar um email para o endereço de email do administrador usando as configurações definidas.
