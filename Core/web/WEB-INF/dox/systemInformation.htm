The Instance ID is automatically assigned to your Mango instance by the Mango Home servers, and should not be changed. It is used to assign rights to documentation among other things. When you upgrade to a commercial version of Mango, you may need to provide this value.

The Database size is the currently amount of storage space consumed by the Mango database. The majority of space is typically taken up by report instances, point readings, and events. To minimize the database size the purging characteristics for each of these items should be set as aggressively as is practical.

Click the icon to have Mango check for new version immediately. Mango will also automatically check periodically. If available upgrades are found, an information alert will be raised. This panel also displays the current version of Mango that is running.
