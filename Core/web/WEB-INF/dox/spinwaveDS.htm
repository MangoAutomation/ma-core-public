Overview

The Spinwave data source is used to gather data from Spinwave devices connected to a local gateway, and accessible via RS232 or RS485 (requires adapter) communication. This data source receives regular updates from the connected devices; it does not poll.

Configuration

Every data source requires a Name, which can be any description. The Port selection provides a list of all available serial ports on the host hardware. This corresponds to the port into which the Spinwave gateway is connected. The Message version depends upon the hardware that you are using. Consult your Spinwave documentation for details.

Sensor addresses are typically found by using the sensor listener functionality described below, although if you know the unique addresses of your hardware you may enter them manually. After entering the address value, click the icon to add it to the list of known addresses.

Spinwave sensor listener

Once the Spinwave gateway has been connected to the host, a listener can be started that waits for messages to arrive. Typically, devices are configured to send messages every 21 seconds, but other settings are possible. To add the addresses of all devices that were heard to the Sensor addresses list, click the "Add found sensors to list" button.
